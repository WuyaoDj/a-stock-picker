# A股智能选股系统

## 功能

- **多因子选股**：结合估值（PE/PB）+ 动量（20日涨幅）+ 量价（量比）
- **自动回测**：验证策略历史表现
- **参数优化**：网格搜索找到最优参数组合

## 安装

```bash
cd stock-picker
pip install -r requirements.txt
```

## 使用

```bash
python stock_picker.py
```

## 策略逻辑

1. **估值因子**：低PE + 低PB（价值股）
2. **动量因子**：20日涨幅 5%-30%（避免追高和太弱）
3. **量价因子**：量比 > 1（有资金关注）
4. **综合打分**：加权排序，选出Top N

## 输出

- 实时选股结果（带评分和推荐等级）
- 回测报告（收益率、最大回撤、夏普比率）
- 参数优化结果
- JSON格式完整数据
